# 编译与运行

+ 无需编译，直接运行 Releases 中的 jar包。
+ 运行环境Windows，jdk版本8

# 注意事项

+ 服务器关闭的情况下UI不能正常运行
+ 无绑定账号不能出牌对战
+ 分辨率固定为1600 X 900
